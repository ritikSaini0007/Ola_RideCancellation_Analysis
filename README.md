# Ola_RideCancellation_Analysis
Ola Ride Cancellation Analysis for Business Growth
Project Overview

This project focuses on analyzing Ola ride booking and cancellation data using Power BI. The dashboard helps understand customer booking behavior, ride cancellation trends, revenue performance, and overall ride success rates. The main objective is to identify factors affecting ride completions and provide insights that can support business growth and improve customer satisfaction.

Dashboard Features
Total Bookings and Booking Value Analysis
Ride Status Breakdown (Successful, Cancelled by Driver, Cancelled by Customer, Driver Not Found)
Ride Volume Trend Analysis
Vehicle Type Performance Analysis
Cancellation Insights
Customer Rating Analysis
Revenue Tracking and Monitoring
Tools Used
Power BI
SQL
Excel
DAX
Business Impact

The analysis helps identify cancellation patterns, monitor operational performance, and improve decision-making. These insights can help reduce ride cancellations, increase successful bookings, improve customer experience, and support overall business growth.

Project Workflow

Data Collection → Data Cleaning → Data Analysis → Dashboard Development → Insight Generation → Business Recommendations.
